# Task 5 — Проектирование технологического стека и расчёт стоимости

## Описание

Технологический ландшафт, экономическое обоснование и стратегический роадмап внедрения Data Mesh для компании «Будущее 2.0».

## Артефакты

| Файл | Описание |
|------|----------|
| `tech-radar.drawio` | Расширенный технический радар с 30+ технологиями и паттернами (Adopt/Trial/Assess/Hold) |
| `tco-analysis.md` | TCO-анализ текущей и целевой архитектуры на 3 года с ROI и точкой окупаемости |
| `roadmap.drawio` | Стратегический роадмап Data Mesh на 36 месяцев с ролями, этапами и бизнес-целями |

## Ключевые результаты TCO-анализа

| Метрика | Значение |
|---------|----------|
| TCO текущей архитектуры (3 года) | 200.8 млн ₽ |
| TCO целевой архитектуры (3 года) | 291.8 млн ₽ |
| Скорректированная разница (с учётом бизнес-выгод) | +4.5 млн ₽ (+2%) |
| Точка окупаемости | ~3.5 года |
| ROI за 5 лет | +35% |
